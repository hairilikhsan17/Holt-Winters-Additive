üìå Bagaimana cara menentukan nilai Œ± = 0.3, Œ≤ = 0.7, Œ≥ = 0.1 ?

Nilai ketiga parameter ditentukan melalui eksperimentasi dan minimalisasi error. Prosesnya adalah:

  • Coba berbagai kombinasi nilai Œ±, Œ≤, Œ≥ antara 0 dan 1 (misalnya 0.1, 0.2, 0.3, ‚Ä¶, 0.9).
  • Hitung MAPE untuk setiap kombinasi menggunakan data training (Bulan 1‚Äì10).
  • Pilih kombinasi yang menghasilkan MAPE terkecil ‚Äî itulah kombinasi terbaik.

Kenapa Œ± = 0.3 (kecil)? ‚Äî Level berubah pelan-pelan; data perbaikan meteran tidak melompat drastis setiap bulan, jadi bobot data baru cukup rendah.
Kenapa Œ≤ = 0.7 (besar)? ‚Äî Tren berubah lebih cepat; perubahan pola durasi perbaikan cukup dinamis sepanjang tahun.
Kenapa Œ≥ = 0.1 (sangat kecil)? ‚Äî Pola musiman sangat stabil dan tidak berubah besar dari periode ke periode, sehingga model mempertahankan pola lama lebih kuat.

Rumus Perhitungan Error

Error
Error = Y‚Çú ‚àí F‚Çú
Absolute Error

APE = (|Y‚Çú ‚àí F‚Çú| √∑ Y‚Çú) √ó 100%
MAPE (Mean Absolute Percentage Error)
MAPE = (Œ£ APE) √∑ Jumlah Periode Testing
